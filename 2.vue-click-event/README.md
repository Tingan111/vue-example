### Vue 基礎語法說明

#### 📌 `ref()` 與 `.value`
在 Vue 中，使用 `ref()` 宣告的變數會回傳一個物件，必須透過 `.value` 來讀寫其實際值。

#### 📌 字串處理
- `split('')`：將字串切割成字元陣列。
- `reverse()`：反轉陣列順序。
- `join('')`：將陣列合併為字串，`''` 表示不加入任何分隔符；若省略參數，預設為逗號 `,`。

#### 📌 點擊事件綁定
- `@click` 是 `v-on:click` 的縮寫。
- `@click="函式"` 可綁定點擊時要執行的函式。
- `@click.prevent` 阻止 HTML 預設行為（如連結跳轉）並執行函式。
